✏️ 使用說明（請假怎麼運作、標記、臨時加課、LINE）
請假（不用先設定 LINE 連結）：點課程 → 按「用 LINE 請假」。一按請假，那堂課就會在課表上標「請假」並畫線，按錯可在詳情「取消標記」。送出方式依裝置不同：
・手機：會打開 LINE 並讓你選老師，訊息(含日期時間)已帶好，點老師→送出。
・電腦：會把訊息複製好，請貼到 LINE 電腦版老師的對話送出（電腦瀏覽器無法自動跳進 LINE，這是 LINE 的限制）。
・想一鍵直接開某位老師的對話：到「管理老師」填那位老師的 LINE 連結，之後按請假就會直接開他的對話，訊息已複製、貼上即可。（LINE 不允許把文字自動塞進別人對話，所以一定要貼一下。）
・老師若有 Email，填了可用「Email 請假」，直接開好信、主旨內文都填好，按寄出最省事。

註記：詳情視窗可幫每堂課寫一句註記（如「改線上」），課表上會出現 📝。

臨時加課：按「＋ 新增課程」或在月曆點某天「＋ 新增單次課程」，最上面可「沿用已建立的課程」一鍵帶入名稱/老師/時間，再自由選「每週固定」或「單次／指定日期」。

選填的老師 LINE 連結：只用在「打開 LINE 討論」想直接開對話時。拿法：請老師到自己的 LINE → 主頁 → 設定 → 個人檔案 → 分享，把連結傳給你；或填他的 LINE ID。沒填也不影響請假。

儲存：改動存在這支手機／瀏覽器，換裝置用「匯出／匯入」搬。
