✏️ 使用說明（新增、聯絡老師、衝堂、最簡單的 LINE 設定）
新增課程：在週曆「空白時間格」點一下，就會帶入那天那個時間開新課程；或按上方「＋ 新增課程」。手機版可按每天標題旁的「＋」。

看／聯絡老師：點週曆上的課程色塊，會跳出該堂課的資訊與聯絡按鈕，可用 LINE、Email 或 FB 聯絡，也能一鍵「請假通知」。

最簡單的 LINE 設定：按「📇 管理老師」，在每位老師的 LINE 欄位「貼上連結」或「直接打 LINE ID」都可以（打 ID 我會自動轉成連結）。連結哪裡拿：打開跟老師的 LINE 對話 → 點他大頭貼 → 右上分享 → 複製連結（長得像 https://line.me/ti/p/...），貼上就完成。Email 直接填信箱、FB 填 Messenger 帳號或主頁網址。

衝堂提醒：同一天時間重疊的課，週曆會把兩塊「不同顏色並排」顯示（白色虛線框 + ⚠），最上方也會列出重疊清單；兩姐妹同時段（要分頭接送）會特別標出來。

資料儲存：修改存在這支手機／瀏覽器裡，換裝置不會自動同步。用「⬇ 匯出備份／⬆ 匯入」搬資料或備份。
