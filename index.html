✏️ 怎麼修改這個課表？
全部資料都在這個檔案最下方的 <script> 裡，分成兩塊，改完存檔 push 到 GitHub 就會更新：

1. 老師（teachers）：填上每位老師的 line。
・個人好友：到 LINE「加好友 → 我的 QR code / 邀請」拿到 https://line.me/ti/p/xxxx 貼上。
・官方帳號：用 https://line.me/R/oaMessage/@帳號ID/? 還能直接帶入文字（最接近自動送出）。
・先不填就留 "#"，請假會改用 LINE 分享選單。

2. 課程（courses）：固定鐘點的填 start／end（24 小時制，如 "09:30"、"14:00"），課表會自動照時間排序；沒固定時間的用 timeNote 寫說明（如「晚餐後 20 分鐘」）。再改 day（週一～週日 / 週末）、status（fixed 固定／flex 彈性／family 家庭／pause 暫緩）。標 pause 的不會出現在今天與週總覽。

請假鈕：按下後會自動把請假訊息複製好、並打開那位老師的 LINE，你貼上送出就完成。
